# 🍪 Food Sales Analytics Dashboard (Power BI)

This project showcases an interactive Food Sales Analytics Dashboard built using Power BI. The dashboard provides insights into sales performance, profit trends, regional distribution, and product category analysis.

---

## 🚀 Features

- 📈 Buying vs Selling Amount Analysis
- 💰 Profit Tracking (Average & Total Profit)
- 🛒 Product-wise Sales Performance
- 🌍 Region-wise Sales Distribution (East vs West)
- 🏙 City-wise Quantity Analysis (Boston, LA, NY, San Diego)
- 📅 Quarterly Performance Insights
- 📊 Interactive Filters & Slicers

---

## 🛠 Tools Used

- Power BI
- Microsoft Excel
- Data Cleaning & Transformation

---

## 📂 Files Included

- `food_sales_data.xlsx` → Dataset used
- `Food_Sales_Dashboard.pbit` → Power BI Template
- `dashboard_preview.png` → Dashboard Screenshot

---

## 📌 Key Insights

- Highest sales observed in **East Region**
- Certain products like **Cookies & Bars** generate higher revenue
- Profit margins fluctuate across quarters
- City-wise demand varies significantly
